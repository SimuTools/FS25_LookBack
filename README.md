# Look Back

Look Back adds a quick rear-view camera function to Farming Simulator 25.

The mod was mainly created for players who like driving from the interior view. It lets you quickly look backwards while reversing, maneuvering or attaching equipment without having to rotate the camera manually every time.

Even though it was designed with the interior camera in mind, the function works globally and can also be used in exterior view.

## How It Works

Hold the configured key to look backwards.

When you release the key, the camera returns to its previous position automatically.

Default key:

```txt
Hold Ctrl + L
```

The key binding can be changed in the game settings:

```txt
Options -> Controls
```
